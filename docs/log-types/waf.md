# AWS WAF Logs

This document describes the AWS WAF Logs and how they are processed by the Cribl Pack. AWS Web Application Firewall (WAF) is a web application firewall that protects web applications from common web exploits, attacks, bots and custom intrusion that may affect availability, compromise security, or consume excessive resources. WAF Logs provide information about all a requests cEvaluated by a web ACL ACLs in AWS WAF. These can be blocked or allowed based on the rules.

## Log Format

AWS WAF Logs are delivered as JSON format!
